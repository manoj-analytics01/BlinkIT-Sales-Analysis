# 🛒 BlinkIT Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes BlinkIT grocery sales using Power BI. The dashboard provides insights into sales performance, customer satisfaction, inventory distribution, and outlet performance to support data-driven business decisions.

---

## 🎯 Business Objective

The goal of this project is to:

- Analyze total sales performance
- Evaluate customer ratings
- Compare outlet performance
- Identify top-performing product categories
- Visualize sales trends using interactive dashboards

---

## 🛠 Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel
- CSV Dataset

---

## 📊 Key Performance Indicators (KPIs)

- Total Sales
- Average Sales
- Number of Items Sold
- Average Customer Rating

---

## 📈 Dashboard Features

- Sales by Fat Content
- Sales by Item Type
- Sales by Outlet Size
- Sales by Outlet Location
- Sales by Outlet Type
- Outlet Establishment Analysis
- Interactive Filters & Slicers

---

## 💡 Key Insights

- Low Fat products contribute significantly to total sales.
- Tier 3 outlets generate the highest revenue.
- Supermarket Type 1 records the highest sales.
- Fruits, Snacks, and Household items are among the top-selling categories.
- Customer ratings remain consistently high across outlet types.
